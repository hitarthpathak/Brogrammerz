About

Brogrammerz is an online social directory of computer programmers around the world. The main goal of the website is to be a centralized platform for accessing both personal and profesional information about programmers. It also has a ranking feature (based on the number of followers), so that you can find out who are the top programmers of the world.

The website is "intentionally" made from scratch, without using any frameworks and libraries, just vanilla HTML, CSS and JavaScript (for front-end) and NodeJS and MondoDb (for backend). There is also not much designing since it's just a directory (as of now).
